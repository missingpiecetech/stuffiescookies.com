# stuffiescookies.com

A website for Stuffie's Cookies - a local cookie baking and delivery service offering big, chewy cookies with less sugar.

## Features

- Homepage with company information and key features
- Dedicated flavors page with detailed cookie descriptions
- Responsive design with mobile support
- SEO-optimized with proper meta tags
- Built with SvelteKit
- FontAwesome icons for visual enhancement

## Development

Install dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

## Building

Create a production build:

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```

## Cookie Offerings

Stuffie's Cookies specializes in:
- Big, chewy cookies
- Reduced sugar recipes
- Fresh baked to order
- Local delivery service
- 9 delicious flavors to choose from
